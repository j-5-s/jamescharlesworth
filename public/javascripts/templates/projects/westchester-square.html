Westchester Square - Condominum Web Portal

PHP, Symfony, Frontend JavaScript, Design

Westchester Square, a condominium in Ansley Park, was looking for a way for residents to access shared documents as well as provide a forward facing web site for their community. This PHP based application provides admin roles to manage the site as well as standard users roles to view content without making changes. Users can login and download association documents, view community contact information, provide notifications to residents, and use contact forms for residents to get in touch with association managers and maintenance contractors.

The Gist

  • PHP Web Application built on Symfony
  • Multiple User Roles
  • Admin's can upload and maintain the documents and contact information
  • Built with PHP and minor Javascript
