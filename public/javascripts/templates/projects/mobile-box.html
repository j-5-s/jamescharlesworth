MobileBox - Mobile jQuery Lightbox Plugin

jQuery, JavaScript, Open Source, UX

Most people have seen the jQuery plugin colorbox in action. It provides a great way to enlarge images in the browser without taking the user to a new page. The downside is that it's the same user experience on a mobile phone. Most of the time the image is not as large as it could be on the users device and closing the popup dialog can be tough with such a small link.

MobileBox is intended to use with colorbox or other lightbox plugins and will override the lightbox effect for mobile users and provide a mobile centric gallery.

The Gist

  • jQuery Plugin
  • Responsive design for mobile applications
  • Touch enabled
